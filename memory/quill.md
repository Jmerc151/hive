# Business Pillar: The Pass (formerly Ember)
Restaurant kitchen management SaaS at thepass.kitchen (sous-frontend.vercel.app).
Built features: AI Recipe Standardizer, Quick Reference Cards, Allergen Matrix, Staff Progress Tracking, AI Quiz Generator, landing page with pricing.
Stack: React 19 + Vite (Vercel) / Express 5 + PostgreSQL (Railway).
Next priorities: Stripe billing, cloud photo uploads, domain purchase, multilingual translation.
